# The Green Gender Gap: Environmental Attitudes and Pro-Environmental Vote Choice Across Europe

Replication code and (derived) data for the article:

> *The Green Gender Gap: Environmental Ideology and Political Behaviour Across Europe*
> https://www.sciencedirect.com/science/article/pii/S026137942600020X

## Repository structure

```
01_Data/
  Ess1-11.R        Combined, recoded dataset (rounds 1-11) produced by 02_Code/p1_code_data.R
  MPD/             Manifesto Project (MPD) reference files used to identify green parties
  greenparty.xlsx  Supplementary reference file
  ESS/             NOT included - see "Getting the ESS data" below

02_Code/
  p1_code_data.R         Builds the combined dataset (01_Data/Ess1-11.R) from the raw ESS files and MPD references
  p1_code_analysis.Rmd   Main analysis (regional estimates, trends over time, mediation analysis)
  p1_code_analysis.pdf   Knitted output of the analysis
  p1_code_appendix.Rmd   Supplementary/appendix analyses
  p1_code_appendix.pdf   Knitted output of the appendix

03_Output/
  Figures/         Figures generated by the analysis scripts
```

## Getting the ESS data

The raw European Social Survey (ESS) data files are **not included** in this
repository, in line with ESS's terms of use (the data must be downloaded by
each user under their own registration).

To reproduce `01_Data/Ess1-11.R` from scratch:

1. Register and download the following round files from the
   [ESS Data Portal](https://ess.sikt.no/en/) (SPSS `.sav` format):

   | Round | File(s) |
   |-------|---------|
   | 1  | `ESS1e06_7.sav` |
   | 2  | `ESS2e03_6.sav` |
   | 3  | `ESS3e03_7.sav` |
   | 4  | `ESS4e04_6.sav` |
   | 5  | `ESS5e03_5.sav` |
   | 6  | `ESS6e02_6.sav` |
   | 7  | `ESS7e02_3.sav` |
   | 8  | `ESS8e02_3.sav` |
   | 9  | `ESS9e03_2.sav` |
   | 10 | `ESS10.sav`, `ESS10SC.sav` |
   | 11 | `ESS11.sav` |

2. Place each file in `01_Data/ESS/ESS round <N>/`, e.g.
   `01_Data/ESS/ESS round 1/ESS1e06_7.sav`.

3. Run `02_Code/p1_code_data.R`. This merges each round with the
   corresponding MPD reference file in `01_Data/MPD/`, recodes green-party
   voting, education, occupational class (Oesch scheme), and a climate
   policy index, and saves the combined dataset to `01_Data/Ess1-11.R`.

   `01_Data/Ess1-11.R` is included in this repository as the derived dataset
   used by the analysis scripts, so step 3 is only needed if you want to
   rebuild it yourself.

## Running the analysis

`02_Code/p1_code_analysis.Rmd` and `02_Code/p1_code_appendix.Rmd` load
`01_Data/Ess1-11.R` and reproduce the tables and figures reported in the
article and its appendix.

**Note:** the scripts currently use absolute file paths
(e.g. `setwd("C:/Users/.../Paper 1")` and
`load("C:/Users/.../Paper 1/01_Data/Ess1-11.R")`). Update these paths to
match your local copy of this repository before running.

## Key variables in `Ess1-11.R`

- `greenvote` - voted for a green party (1) vs. another party (0)
- `class8` / `class16` - occupational class (Oesch class scheme)
- `eisced2` - harmonized education level across countries
- `polindex` - climate policy support index (ESS round 8 only)
- `region` - country grouping (West / Central-Eastern / South / North)
